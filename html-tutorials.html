Lesson 1: Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create webpages. It tells the browser how to structure content like text, images, and links.

HTML files end with .html and use tags like <h1>, <p>, and <a> to define elements.

Lesson 2: HTML Tags

Tags are used to mark up the start and end of elements in HTML. Most tags come in pairs like <p> and </p>.

Some important tags:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <br> - Line break (self-closing)
  • <hr> - Horizontal rule (self-closing)
  • <a> - Hyperlink
  • <img> - Image (self-closing)
  • <div> and <span> - Grouping elements
